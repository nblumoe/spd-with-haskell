renderCountdown :: Countdown -> String #

Produce visual representation of a countdown

Examples:

>>> renderCountdown NotStarted
"-----\n| > |\n-----"

>>> renderCountdown Completed
"------\n| :D |\n------"

>>> renderCountdown $ Count 10
"------\n| 10 |\n------"

>>> renderCountdown $ Count 3
"-----\n| 3 |\n-----"
